# Käyttöohje

Projektin viimeisin [release](https://github.com/Kissaniemi/ot-harjoitustyo/releases/tag/viikko6)

## Ennen käynnistystä

### Asenna ensin ohjelman riippuvuudet:

poetry install

### Käynnistä ohjelma:

poetry run invoke start
